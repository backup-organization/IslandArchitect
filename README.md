# IslandArchitect

### A plugin to create custom skyblock island generators.

### Features:

 - Helpful to server builders
 - Random generation regex for blocks
 - Regex editor form and inventory UI

### How to setup?

 - Put the plugin in your plugins folder.
 - Restart and enjoy.
